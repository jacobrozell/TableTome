Get help

Tabletome v1.0 is an offline companion for Age of Sigmar: Spearhead and Warhammer 40,000 (11th Edition) — game guides, rules reference, Guided Match with battle tracking, optional model collection, and match history. Everything runs on your device; no account required.

Report a bug or request a feature

Open an issue on GitHub with your iOS version, device model, and steps to reproduce: github.com/jacobrozell/TableTome/issues

Common questions

  • Offline use — Play and Rules work without network access after install.
  • First game — open the Play tab, use the new-player chooser or pick a game under All games, then follow Getting Started and Guided Match.
  • Replay onboarding — Settings → App Tour → View App Tour.
  • Battle tracker tips — Settings → App Tour → Replay Battle Tracker Tips.
  • Collection sample data — after your first visit to Play, the Models tab may offer sample armies to explore; delete them anytime.
  • Nearby sync — optional in Guided Match when both armies are set; both players need Tabletome on the same Wi‑Fi or Bluetooth range. You can also paste a match code if nearby sync is unavailable.
  • Official rules — Tabletome explains mechanics in original wording for learning; always defer to Games Workshop publications for tournaments.
