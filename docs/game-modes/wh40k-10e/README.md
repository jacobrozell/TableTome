# Warhammer 40,000 — 10th Edition (full matched play)

**Game system id:** None — `wh40k-10e` is a legacy persisted id only  
**Status:** **Not planned** for Tabletome

## What we ship instead

| Player need | Use this |
|-------------|----------|
| Combat Patrol starter box | [`wh40k-10e-cp`](../combat-patrol/) — **10th Edition patrol rules** |
| Current full 40k (Armageddon, Battleforce, lists) | [`wh40k-11e`](../wh40k-11e/) — **11th Edition** |

Full 10e matched play (points, detachments, Crusade, all factions) is **explicitly out of scope** per [PRODUCT_SCOPE.md](../PRODUCT_SCOPE.md). Do not imply full 10e support in UI copy.

## Why this folder exists

- Edition isolation: 10e patrol (CP) vs 11e full vs deferred 10e matched play.
- `Wh40k10eCombatRollResolution` serves **Combat Patrol only** — not a full 10e game mode.
- Legacy `wh40k-10e` may appear behind `-enable_full_product_surface` for plumbing tests only.
