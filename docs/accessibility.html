Our commitment

Tabletome targets WCAG 2.1 AA engineering practices on core v0.1 flows: Learn (Getting Started), Rules reference, and Settings.

Built-in iOS accessibility

  • VoiceOver — labels, hints, and identifiers on interactive controls in Learn, Rules, and Settings.
  • Dynamic Type — semantic text styles scale with your system font size.
  • Touch targets — minimum 44×44 pt on primary controls.
  • Reduce Motion — guide step completion animations respect the system setting.
  • Dark Mode — system, light, and dark appearance in Settings.

Known limitations (v0.1)

  • English only.
  • Manual VoiceOver audit pending before App Store release.
  • External PDF and web links open in Safari / system viewers outside the app.

Feedback

If you encounter an accessibility barrier, please file an issue with the screen name and iOS version: GitHub Issues
