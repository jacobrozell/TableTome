Get help

Tabletome v0.1 focuses on the Age of Sigmar: Spearhead Getting Started walkthrough and offline rules reference.

Report a bug or request a feature

Open an issue on GitHub with your iOS version, device model, and steps to reproduce: github.com/jacobrozell/Tabletome/issues

Common questions

  • Offline use — Learn and Rules tabs work without network access after install.
  • Lost guide checkmarks — progress is stored on-device only; use Settings → Reset Guide Progress to start over.
  • Official rules — Tabletome summarizes Spearhead play; always defer to Games Workshop publications for tournaments.
